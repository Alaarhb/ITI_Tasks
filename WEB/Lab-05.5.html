Part 2: Number Object Comparison Discussion

Statement A:

var t1 = new Number(10);
var t2 = new Number(10);
console.log(t1 == t2);

Statement B:

console.log([1,2,3] == [1,2,3]);

Discussion Results:

Statement A Result: false

Why: When you create Number objects with new Number(), you create separate object instances. Even though they contain the same value (10), they are different objects in memory. Object comparison (==) compares references, not values.

Statement B Result: false

Why: Similar to Number objects, arrays are objects. Two arrays with identical contents are still different objects in memory. Array comparison compares references, not the array contents.

Key Learning: In JavaScript, objects (including Number objects and arrays) are compared by reference, not by value. Even if they have identical content, they are considered different if they are separate instances.
